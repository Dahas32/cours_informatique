Auteurs :

  • Emmanuel Daucé

Cours en duo. Nous ne montrerons ici que les trois cours de François Brucker.

Vous aurez besoin d’une installation de jupyter notebook opérationnelle sur votre ordinateur. Une façon simple d’obtenir ça est d’installer anaconda, qui est une distribution python à visée data science. Pour cela, vous pouvez suivre ce tutoriel.

  1. téléchargez le cours 1
  2. téléchargez le cours 2
  3. téléchargez le cours 3

Les cours sont téléchargés sous la forme d’un dossier compressé au format zip. Si vous n’avez pas de logiciel pour dézipper des fichiers et que vous êtes sous windows, vous pouvez installer 7zip (choisissez 64-bit Windows x64 comme système).
