Cours/Algorithmie/Écrire du Code

Un programme est un algorithme qui ne s'arrête pas forcément. On les décrit en algorithmie avec du pseudo-code mais lorsque l'on veut exécuter ses programmes sur un ordinateur, on est obligé de transcrire le pseudo-code dans un langage de programmation (on utilisera le python dans ce cours) puis de l'exécuter sur un ordinateur.

Nous allons voir dans cette partie qu'écrire du code ou du pseudo-code est équivalent.
