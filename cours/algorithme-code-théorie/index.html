Cours/Algorithme, code et théorie
Prérequis :
  • /cours/utiliser-python/

Ce cours montrera l'informatique sous trois aspects complémentaires — théorie, code et algorithmes — que tout honnête informaticien devrait connaître. Il s'adresse à des personnes ayant des connaissances minimales en informatiques mais voulant (ou étant obligé d' :-)) approfondir le sujet.

Prérequis

Les connaissances et les outils que vous devez avoir pour commencer le cours. Ils sont minimaux :

  • vous allez coder. Et beaucoup. Pour cela, il faut que vous ayez un ordinateur configuré pour le faire.
  • connaissances système :
    • pouvoir ouvrir un explorateur de fichier
    • installer un programme avec un installeur

indispensable !

Préparez votre ordinateur pour le développement en suivant ce tutoriel : un ordinateur pour le développement.

Parties

Ce cours est composé de trois grandes parties qui s'enchevêtrent.

  • Algorithme
    • Nombres Aléatoires
      • Registres à décalage
      • Générer des nombres
    • Algorithmes de calcul d'enveloppe convexe
  • Théorie
    • Nombres pseudo-aléatoires
