Initialisation du projet git.

Prérequis

Suivez la partie Création du compte github du cours gestion des sources.

Vous être prêt à utiliser git.

Vous pourrez lire le cours gestion des sources et les différents projets pour vous familiariser un peu plus avec la gestion des sources.

Initialisation du projet

On va utiliser l'outil desktop de github.

Une fois l'application lancée, connectez vous, puis créez un nouveau projet (File > New repository) : choisissez son nom (numerologie), puis cliquez sur publish repository.
