installation

sous Linux

A priori déjà installé

sous Mac

On l’installe avec brew :

brew install openssh
sous Windows

Les dernières versions de windows viennent avec tout ssh d’installé. Il faut juste faire en sorte que l’Agent ssh soit lancé au démarrage (on suit le tuto de gestion des clés OpenSSH). Ouvrez un fenêtre powershell en mode administrateur (clique droit sur le drapeau, puis choissser pouwershel (admin)), puis tapez les commandes :

# Set the sshd service to be started automatically
Get-Service -Name ssh-agent | Set-Service -StartupType Automatic

# Now start the sshd service
Start-Service ssh-agent

L’agent fonctionne différemment sous windows que sous un système unix. Mais normalement, il devrait être reconnu, en particulier par git.
