## Structure du projet

Le nom du dossier du projet est `numérologie`{.fichier}. Sa structure finale est :

```text
.
├── back
│   └── numérologie.js
├── index.js
├── package.json
└── static
    ├── index.html
    └── main.css
```

{% note %}
Ni le dossier `node_modules` ni le fichier `package-lock.json`{.fichier} n'ont été représentés.
{% endnote %}
