## Structure du projet

Le nom du dossier du projet est `numérologie`{.fichier}. Sa structure finale est :

```text

├── site
│   ├── index.html
│   ├── main.css
│   └── numérologie.js
├── tests
│   ├── mes_tests.js
│   └── tests_numérologie.js
├── todos
│   └── todos.md
└── user-stories

    └── url-numéro.md
```
